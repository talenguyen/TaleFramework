# TaleFramework
This is a sample project to demo how to model Android Project follow MVP pattern.

# Requirement
Android Studio 1.1.0

Android SDK: 21

JDK: 1.8

Android Build Tool: 21.1.1

1. Appcompat v7
2. RecyclerView v7
3. CardView v7
4. Retrolambda
5. Dagger
6. Retrofit
7. Gson
8. ButterKnife
9. Timber
10. RxAndroid
11. Picasso
